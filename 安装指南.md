# QuickClass 安装指南

## 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | macOS、Windows、Linux |
| Node.js | 18.17 或更高版本 |
| npm | 9.x 或更高版本 |

确认已安装 Node.js：

```bash
node --version   # 应显示 v18.17.0 或更高
npm --version    # 应显示 9.x 或更高
```

---

## 安装步骤（在线安装）

### macOS / Linux

**第一步：启动服务**

在项目根目录下执行：

```bash
./start.sh
```

等待启动完成后，浏览器会自动打开 `http://localhost:3000`。

**第二步：停止服务**

```bash
./stop.sh
```

---

### Windows

**第一步：启动服务**

双击项目目录下的 `start.bat`。

等待启动完成（约 1-2 分钟），浏览器会自动打开。

**第二步：停止服务**

双击 `stop.bat`。

## 首次使用流程

### 第一步：注册教师账号

1. 访问 `http://localhost:3000`
2. 点击「教师注册」
3. 填写邮箱、密码、姓名
4. 完成注册

### 第二步：配置 AI 服务

1. 进入「系统设置」
2. 填写 AI Base URL（如 `https://dashscope.aliyuncs.com/compatible-mode/v1`）
3. 填写 API Key
4. 选择 AI 模型（如 `qwen-turbo`）
5. 点击「测试连接」确认配置正确
6. 点击「保存配置」

### 第三步：创建班级

1. 进入「班级管理」
2. 点击「创建班级」
3. 填写班级名称、自定义邀请码
4. 保存后记录邀请码

### 第四步：创建课堂

1. 进入「课堂管理」
2. 点击「创建课堂」
3. 填写基本信息、添加学习活动
4. 分配班级
5. 点击「启用」

### 第五步：学生加入

1. 学生访问 `http://<教师电脑IP>:3000`
2. 点击「学生入口」
3. 输入姓名和邀请码
4. 开始学习

## 常见问题

**Q: 启动脚本报错"权限被拒绝"**
A: macOS/Linux 上执行：`chmod +x start.sh stop.sh`

**Q: Windows 上启动后浏览器未自动打开**
A: 手动打开浏览器，访问 `http://localhost:3000`

**Q: 学生无法访问教师电脑上的服务**
A: 确保教师电脑防火墙允许 3000 端口，学生使用局域网 IP 访问（如 `http://192.168.1.100:3000`）

**Q: AI 无法回答问题**
A: 检查「系统设置」中 API Key 是否正确，尝试点击「测试连接」

**Q: 如何备份数据**
A: 停止服务后，复制 `prisma/dev.db` 文件到安全位置

npm run dev        # 开发模式启动（不推荐生产使用）

npm start         # 生产模式启动
